# THE_NOXA_TOOL v1.0.0
**The Ultimate OSINT & Security Testing Suite**

THE_NOXA_TOOL is a multi-functional suite written in Python, designed for security researchers, OSINT experts, and cybersecurity enthusiasts. It combines offensive, defensive, and digital investigation tools within a single command-line interface.

---

## Technical Features
The tool is divided into 10 operational modules:

1. **Dos Attack**: Stress testing for servers and network infrastructures.
2. **Proxy Support**: (Under Development) Anonymity and traffic routing management.
3. **Malware Builder**: Automated generation of files for penetration testing (Requires Administrator privileges).
4. **IP Grabber**: Generation of tracking links for IP address identification.
5. **BruteForce Cracker**: Automated credential testing via computational power.
6. **Dictionary Cracker**: Password recovery and testing based on known wordlists.
7. **DDoS (Premium)**: Advanced and distributed version of the Denial of Service module.
8. **Nickname Scanner**: Cross-platform OSINT search across 50+ social media platforms and websites.
9. **Phone Lookup**: Metadata extraction and geographical information from international phone numbers.
10. **Email Lookup**: OSINT investigation and account verification for email addresses.
